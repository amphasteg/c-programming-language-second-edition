### Compilation
#### Unix-Based Systems (Linux, MacOS, etc.)
To compile on a Unix-based system, you can type the command `cc program.c`, where "program" is the file name of the C program you are attempting to compile.

After compilation, you should be able to run the program using the command `./a.out`.

#### Windows
To compile for Windows, you will need either **Visual Studio** or **the build tools for Visual Studio** installed on your system as well as the **Desktop development with C++ workload**.

You will also need the **MSVC compiler**.

After obtaining these, you will need to open developer command prompt, cd into the directory your C file is located, and compile it using the command `cl program.c`, where "program" is the name of the C program you are attempting to compiled.

After compiling, you can execute your program by typing the name of it followed by the extension `.exe`. For example: `program.exe`. 

More detailed information can be found on [Microsoft's website](https://learn.microsoft.com/en-us/cpp/build/walkthrough-compile-a-c-program-on-the-command-line?view=msvc-170).
